      Gemstone Namespace

      The Gemstone namespace organizes all common Gemstone library functionality. The root namespace also includes classes, e.g., BigEndian, EventArgsT, Encoding7Bit and Common.

      Structures

      ShortTime Represents a high resolution time that is very granular but may drift if trying to accurately measure long time durations (Such as hours). This time is not adjusted with changes to the system clock. Typical clock drifts by about 2-3 ms per minute as apposed to 0.4ms per minute for standard DateTime.
      Ticks Represents an instant in time, or time period, as a 64-bit signed integer with a value that is expressed as the number of 100-nanosecond intervals that have elapsed since 12:00:00 midnight, January 1, 0001.
      TimerCapabilities Represents information about the system's multimedia timer capabilities.
