      Remarks

      This class behaves just like a Double representing a charge in coulombs; it is implicitly castable to and from a Double and therefore can be generally used "as" a double, but it has the advantage of handling conversions to and from other charge representations, specifically ampere-hours, abcoulomb (a.k.a., an electromagnetic unit), statcoulomb (a.k.a., electrostatic unit or franklin), atomic unit of charge and faraday. Metric conversions are handled simply by applying the needed SI conversion factor, for example:

      Example

      Convert charge, in coulombs, to kilocoulombs:
      C#
      public double GetKilocoulombs(Charge coulombs)
{
    return coulombs / SI.Kilo;
}
