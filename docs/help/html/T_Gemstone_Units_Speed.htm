      Remarks

      This class behaves just like a Double representing a speed in meters per second; it is implicitly castable to and from a Double and therefore can be generally used "as" a double, but it has the advantage of handling conversions to and from other speed representations, specifically miles per hour, kilometers per hour, feet per minute, inches per second, knots and Mach.

      Example

      This example converts mph to km/h:
      C#
      public double GetKilometersPerHour(double milesPerHour)
{
    return Speed.FromMilesPerHour(milesPerHour).ToKilometersPerHour();
}
