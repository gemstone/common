s  String
A string containing a number to convert.
result  Charge
When this method returns, contains the Charge value equivalent to the number contained in s, if the conversion succeeded, or zero if the conversion failed. The conversion fails if the s parameter is null, is not of the correct format, or represents a number less than MinValue or greater than MaxValue. This parameter is passed uninitialized.

Return Value

Boolean
true if s was converted successfully; otherwise, false.
